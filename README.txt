Instalação e Uso

Para usar o sistema, é necessário ter o PHP e o MySQL instalados em sua máquina. Em seguida, siga as etapas abaixo:

Clone este repositório: git clone https://github.com/seu-usuario/nome-do-repositorio.git
Crie um banco de dados MySQL e importe o arquivo "database.sql" para criar as tabelas necessárias.
Recomendo baixar o XAMPP
Altere as configurações de conexão do banco de dados no arquivo "config.php".
Abra o arquivo "index.php" em seu navegador para acessar o sistema.

Funcionalidades
O sistema possui as seguintes funcionalidades:

Cadastro de empresas e alunos;
Login para empresas e alunos;
Cadastro de vagas de emprego e estágio;
Pesquisa de vagas por área de atuação;
Candidatura às vagas pelos alunos;
Gerenciamento de candidaturas pelas empresas.
Sistema de perfil para empresa e aluno
